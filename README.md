# Tamagotchi_UFO

Context:
작고 귀여운 우주 아기 다마고치 앱을 만들고 있어.

Task:
홈 화면과 상태 시스템을 먼저 구현해줘.

Requirements:
배고픔, 목마름, 청결도, 기분, 에너지 상태가 있어야 하고,
먹이기/물주기/화장실/청소/잠재우기 버튼이 있어야 해.

Design:
파스텔톤, 둥근 카드, 작은 우주 파티클, 귀엽고 감성적인 느낌.

Constraints:
모바일 우선, 복잡한 기능 제외, 테스트 가능한 MVP로.

Output:
작동하는 홈 화면과 상태 변화 로직.


#####
1. Lovable 첫 프롬프트로 전체 구조 생성
2. 홈 화면 + 상태 시스템 먼저 완성
3. 먹이 / 물 / 화장실 / 청소 / 수면 기능 추가
4.캐릭터 감성 표현 다듬기
5. 데이터 저장 / 기록 기능 추가
6. 온보딩 / 앱 이름 / 문구 / 아이콘 컨셉 정리
7. 웹앱 형태로 먼저 공개
8. 필요하면 그다음 앱 패키징/스토어 배포 검토
